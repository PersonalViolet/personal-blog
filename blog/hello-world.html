个人博客上线了

2026-06-10博客Next.js前端

关于这个博客

这是一个使用 Next.js 构建的个人博客网站。设计上追求简洁与专注的阅读体验，同时融入了一些个人特色。

技术栈

  • 框架: Next.js 16 (App Router)
  • 语言: TypeScript
  • 样式: Tailwind CSS
  • 内容: Markdown 文件
  • 主题: 支持深浅色切换

设计理念

网站采用了 紫罗兰色 作为主题色，VIOLET 既是我的名字，也是这个网站的色彩基调。

"Do more, say less" — 少说多做，这是我的信条。

主要功能

  1. 项目展示 — 从 GitHub API 自动拉取仓库信息，分为精选展示和一般展示
  2. 博客观看 — 所有文章以 Markdown 格式存储在 content/blog/ 目录下
  3. 深浅色切换 — 支持浅色与深色两种主题，可手动切换
  4. 响应式设计 — 适配桌面端与移动端

未来计划

  • RSS 订阅支持
  • 评论系统
  • 全文搜索

如果你对这个博客有任何建议，欢迎在 GitHub 上联系我。
